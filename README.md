# 🌦 Weather App – A Simple React Project  

This **Weather App** is a user-friendly application built with **React.js** and **Tailwind CSS** that provides real-time weather updates. Users can search for any city and get **temperature, humidity, wind speed, and weather conditions** instantly.  

---

## 🚀 Features  

✅ **City-Based Weather Search** – Fetches live weather data for any location.  
✅ **API Integration** – Uses the OpenWeatherMap API for real-time weather updates.  
✅ **State Management** – Utilizes `useState` and `useEffect` hooks to handle API calls and data updates.  
✅ **Dynamic UI Updates** – Weather details update instantly based on user input.  
✅ **Error Handling** – Displays alerts for invalid city names or API errors.  
✅ **Responsive Design** – Fully optimized for desktops and mobile devices using Tailwind CSS.  

---

## 🛠 Tech Stack  

- **Frontend:** React.js, Tailwind CSS  
- **API:** OpenWeatherMap API  
- **State Management:** useState, useEffect Hooks
